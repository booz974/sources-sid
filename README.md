## Règles fondamentales

1. **Ne pas uploader de PDF** — Ce repo est un inventaire de métadonnées, pas une bibliothèque de fichiers.
2. **Acquérir légalement** — Chaque institution/apprenant doit acquérir ses propres copies selon les droits applicables.
3. **Citer avec l'ID canonique** — Utiliser toujours l'ID canonique fourni (ex: `[pettersson-id5-cognition]`) dans tous les documents du curriculum.
4. **Hiérarchie des sources** — `sid_foundation.md` > `sid_body_of_knowledge.md` > `sid_standard.md` > `sid_position_paper.md` > `sid_manifeste.md` > `SID_Foundational_Paper_v0_1_2.md`
